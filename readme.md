# Real-Time-Stock-Prediction

A real-time stock prediction system that loads data from stock exchange APIs, stores it in Azure Data Warehouse, forecasts returns using Gradient Boosting models, and displays results through a React.js dashboard.

## Overview

This project implements an end-to-end pipeline for stock market analysis and prediction:
- **Data Collection**: Fetches real-time data from stock exchange APIs
- **Data Storage**: Stores historical and real-time data in Azure Data Warehouse
- **Forecasting**: Uses Gradient Boosting algorithms to predict stock returns
- **Visualization**: Displays predictions and analytics through a React.js dashboard
